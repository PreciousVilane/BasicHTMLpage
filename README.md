# BasicHTMLpage
This project is a static website created using HTML.  The purpose of this project is to demonstrate how to deploy a static website using GitHub Pages.
# Technologies Used
- HTML
- GitHub Pages
# Deployment
The website was deployed using GitHub Pages by following these steps:
1. A public GitHub repository was created and initialised with a README file.
2. The static website files were uploaded to the repository.
3. GitHub Pages was enabled from the repository settings using the main branch.
4. A live link was generated by GitHub Pages.
